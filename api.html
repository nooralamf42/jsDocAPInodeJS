API Documentation

How to Use the API

The API provides the following endpoints:

  • /api/js - Get information about JavaScript
  • /api/js/{element} - Get information about a specific JavaScript element
  • /api/js/{element}/{method} - Get information about a specific method of a JavaScript element

Example Usage

To retrieve information about JavaScript:

GET /api/js

To retrieve information about the array element:

GET /api/js/array

To retrieve information about the push method of the array element:

GET /api/js/array/push

The API returns JSON objects containing the requested information.
